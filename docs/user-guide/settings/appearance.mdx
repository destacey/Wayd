# Appearance

Wayd lets you personalize the look and feel of the application. Your preferences are saved to your account and follow you across devices.

## Opening the Theme Manager

Click your **avatar** in the top-right corner and select **Theme** from the dropdown. The Theme Manager drawer opens on the right side of the screen.

## Mode

Choose from the available display modes:

| Mode | Description |
|------|-------------|
| **Light** | Bright background, dark text. Best for well-lit environments. |
| **Slate** | Mid-tone grey background. A comfortable middle ground between light and dark. |
| **Dark** | Dark background, light text. Best for low-light environments. |
| **Cartoon** | Playful rounded UI with illustrated background and warm tones. |
| **Shadcn** | Neutral, minimal style inspired by shadcn design tokens. |
| **Glass** | Frosted, translucent visual style with soft depth. |
| **Geek** | High-contrast neon-on-dark style. |
| **Illustration** | Hand-drawn inspired style with bold outlines and pastel accents. |

## Primary Color

Select an accent color from the preset palette. This color is used for buttons, links, active states, and — in Light and Slate modes — the top navigation bar.

The available colors are drawn from Ant Design's standard palette.

Some themes are intentionally style-locked and do not expose primary color customization. In those themes, the primary color section is hidden.

## Density

Controls the spacing and size of UI elements:

| Option | Description |
|--------|-------------|
| **Default** | Standard spacing — recommended for most users. |
| **Compact** | Reduced spacing — useful on smaller screens or when you want more content visible at once. |

## Resetting

Click **Reset to Defaults** to clear all customizations and return to the selected mode's default primary color and default density. Your current mode selection is preserved.

:::note
Changes are saved automatically.
:::
